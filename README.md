# FQsql

`FQsql` أداة سطر أوامر لفحص تطبيقات الويب عبر عدة أوضاع تشغيل.

## المزايا

- فحص تلقائي شامل
- فحص `sql`
- فحص `XSS`
- فحص `IDOR`
- فحص تجاوز التحقق
- بصمة الهدف
- دعم البروكسيات والكوكيز
- تشغيل من سطر الأوامر أو من قائمة تفاعلية
- دعم ويندوز وماك ولينكس

## التوافق

تعمل الأداة على ويندوز وماك ولينكس، لأن بنيتها مكتوبة بشكل مباشر وخفيفة الاعتماديات، بدون ربطها بمكونات خاصة بنظام واحد.

## المتطلبات

- بايثون `3.14`

## التشغيل

```sh
python3 sql.py
python3 sql.py --help
python3 sql.py --menu
```

## أمثلة

### فحص تلقائي

```sh
python3 sql.py -u 'https://example.com/item?id=1' -t auto
```

### فحص `sql`

```sh
python3 sql.py -u 'https://example.com/item?id=1' -t sqli
```

### فحص `XSS`

```sh
python3 sql.py -u 'https://example.com/search?q=test' -t xss
```

### فحص `IDOR`

```sh
python3 sql.py -u 'https://example.com/api/user?id=12' -t idor
```

### فحص تجاوز التحقق

```sh
python3 sql.py -u 'https://example.com/login' -t auth
```

### بصمة الهدف

```sh
python3 sql.py -u 'https://example.com' -t fp
```

## الخيارات

- `-u` الرابط
- `-p` الحمولة
- `-m` الطريقة
- `-P` اسم المتغير
- `-t` نوع الفحص
- `-f` ملف روابط
- `--menu` تشغيل القائمة
- `--proxy` بروكسي
- `--proxy-file` ملف بروكسيات
- `--delay` مهلة بين الطلبات
- `--no-delay` إلغاء المهلة
- `--cookies` كوكيز
- `--fingerprint` بصمة متصفح

## ملاحظات

- الأداة تدعم الفحص المباشر على رابط واحد أو عدة روابط من ملف.
- فيه أوضاع فحص مستقلة حسب الحاجة.
- فيه دعم لتعديل البصمة وإعدادات الطلبات أثناء التشغيل.

## تنبيه

الاستخدام يكون على الأنظمة أو التطبيقات المصرح لك بفحصها فقط.
